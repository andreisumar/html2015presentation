What is HTML

  • HyperText Markup Language
  • Standard markup language used to create web pages
  • HTML documents are described by tags
  • HTML files are interpreted by browsers to render the visible web pages

The History of HTML

В 1980 году британский ученый Тим Бернерс-Ли, сотрудник Европейского института физики частиц (CERN) в Женеве, разработал прототип ENQUIRE - системы обмена документами для сотрудников CERN. В ENQUIRE документы были представлены в виде страниц, которые назывались карточками (cards), между ними можно было перемещаться при помощи ссылок. Многие идеи, использованные в Enquire, спустя 9 лет нашли своё применение во всемирной паутине:

  • двухсторонние гиперссылки (используются, например, в Wikipedia и MediaWiki)
  • редактирование документов на сервере (как в современных CMS)
  • документы хранились в базе данных

В конце 1990 года Бернерс-Ли завершил работу над спецификацией HTML и разработал браузер, поддерживающий этот язык, и необходимое серверное ПО. В том же году он обратился в CERN с запросом о финансировании проекта, однако получил отказ.

В 1991 году он создал документ названный «HTML Tags», в котором предложил 18 элементов для создания веб-страниц. Идея использовать теги, заключенные между угловыми скобками, не его, такие теги уже были в формате SGML (Standard Generalized Markup Language). Тим понимал, что создавать новые стандарты нужно на базе существующих, а разработчики HTML5 и сейчас в полной мере следуют этому принципу. Кроме тега, соответствующего гиперссылкам, предложенные им элементы были в большой мере позаимствованы из SGMLguid - внутреннего формата документации в CERN. 11 тегов из «HTML Tags» используются и по сей день.

Стандарта HTML 1 никогда не было, первой официальной спецификацией стал HTML 2.0, опубликованный IETF (Internet Engineering Task Force). Многие элементы в этой спецификации были заимствованы из существующих реализаций, например, тег уже поддерживался, лидировавшим тогда браузером Mosaic. Позже на смену IETF пришел W3C (World Wide Web Consortium) и следующие версии стандарта HTML публиковались на www.w3c.org.

В версии HTML 4 произошла некоторая «очистка» стандарта. Многие элементы были отмечены как устаревшие и нерекомендованные (В частности, элемент font). Вместо них рекомендовалось использовать таблицы стилей CSS.

Во всемирной паутине долгое время использовались стандарты HTML 4.01, XHTML 1.0 и XHTML 1.1. Веб-страницы на практике оказывались свёрстаны с использованием смеси особенностей, представленных различными спецификациями, включая спецификации программных продуктов, например веб-браузеров, а также сложившихся общеупотребительных приёмов.
Web Hypertext Application Technology Working Group (WHATWG) начала работу над новым стандартом в 2004 году, когда World Wide Web Consortium (W3C) сосредоточился на будущих разработках XHTML 2.0, а HTML 4.01 не изменялся с 2000 года. В 2009 году W3C признал, что срок работы у рабочей группы XHTML 2.0 истёк, и решил не возобновлять его. Впоследствии W3C и WHATWG совместно разрабатывали HTML5.
HTML5 был создан как единый язык разметки, который мог бы сочетать синтаксические нормы HTML и XHTML. Он расширяет, улучшает и рационализирует разметку документов, а также добавляет единый API для сложных веб-приложений. В HTML5 реализовано множество новых синтаксических особенностей. HTML Elements Structure

Sample HTML File


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p<This is a website.</p>
    </body>
</html>

XHTML


<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html lang="EN" xml:lang="EN" xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="content-type" content="text/xml; charset=utf-8" />
<title>Hello World</title>
</head>
    <body>
        <h1>Hello World</h1>
        <p>This is a website.</p>
    </body>
</html>

  • <img src="/images/xhtml.gif" >
  • <p>This paragraph is written according to XHTML syntax.</p>
  • <img src="/images/xhtml.gif" />
  • <img src="/images/xhtml.gif" width=250 height=50 />
  • <img src="/images/xhtml.gif" width="250" height="50" />

  • <option selected="selected">
