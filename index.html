HTML

Что такое HTML

  • Стандартный язык разметки веб-страниц
  • HTML-документы представляют собой набор тегов: 
    <strong></strong>
  • Интерпретируется браузерами для создания видимого представления страниц
  • В настоящее время используется в том числе в качестве графического интерфейса JavaScript-приложений

История HTML

Пример HTML-файла


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p<This is a website.</p>
    </body>
</html>

XHTML


<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html lang="EN" xml:lang="EN" xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="content-type" content="text/xml; charset=utf-8" />
<title>Hello World</title>
</head>
    <body>
        <h1>Hello World</h1>
        <p>This is a website.</p>
    </body>
</html>
