What is HTML

  • HyperText Markup Language
  • Standard markup language used to create web pages
  • HTML documents are described by tags
  • HTML files are interpreted by browsers to render the visible web pages
Язык гипертекстовой разметки, служащий для создания веб-страниц. Состоит из тэгов и интерпретируется браузерами для отображения вебстраниц.

The History of HTML

В 1980 году британский ученый Тим Бернерс-Ли разработал прототип ENQUIRE - системы обмена документами для сотрудников CERN. В ENQUIRE документы были представлены в виде страниц, которые назывались карточками (cards), между ними можно было перемещаться при помощи ссылок.

Многие идеи, использованные в Enquire, спустя 9 лет нашли своё применение при формировании языка HTML.

В конце 1990 года Бернерс-Ли завершил работу над спецификацией HTML и разработал браузер, поддерживающий этот язык.

В 1991 году он создал документ названный «HTML Tags», в котором предложил 18 элементов для создания веб-страниц.

Стандарта HTML 1 никогда не было, первой официальной спецификацией стал HTML 2.0. Многие элементы в этой спецификации были заимствованы из существующих реализаций.

В версии HTML 4 произошла некоторая «очистка» стандарта. Многие элементы были отмечены как устаревшие и нерекомендованные. Вместо них рекомендовалось использовать таблицы стилей CSS.

С 28 октября 2014 года W3C официально рекомендует использовать HTML5 — это значит, что стандарт окончательно финализирован и готов к широкому использованию.

Sample HTML Document


<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
    </head>
    <body>
        <h1>Hello World</h1>
        <p>This is a website.</p>
    </body>
</html>

XHTML


<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
 "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html lang="EN" xml:lang="EN" xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="content-type" content="text/xml; charset=utf-8" />
<title>Hello World</title>
</head>
    <body>
        <h1>Hello World</h1>
        <p>This is a website.</p>
    </body>
</html>

Язык XHTML является более строгим вариантом HTML, он следует всем ограничениям XML и, фактически, XHTML можно воспринимать как приложение языка XML к области разметки гипертекста.

HTML Document Structure

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Hello World</title> 
    </head>
    <body>
        <h1>Hello World</h1>
        <p>This is a website.</p>
    </body>
</html>
  • doctype - Предназначен для указания типа текущего документа. Это необходимо, чтобы браузер понимал, как следует интерпретировать текущую веб-страницу, поскольку HTML существует в нескольких версиях
  • html - корневой элемент страницы. является контейнером, который заключает в себе все содержимое веб-страницы
  • head - Предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными. Содержимое тега <head> не отображается напрямую на веб-странице. (meta, link, title)
  • body - предназначен для хранения содержания веб-страницы
